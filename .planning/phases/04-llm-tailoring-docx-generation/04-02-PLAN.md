**LLMProvider Protocol** (runtime_checkable):
```python
@runtime_checkable
class LLMProvider(Protocol):
    async def complete(
        self,
        system: list[dict],
        messages: list[dict],
        max_tokens: int,
        temperature: float = 0.3,
    ) -> LLMResponse: ...
```

**AnthropicProvider class:**
- `__init__(self, api_key: str, model: str = "claude-sonnet-4-5")`: Create AsyncAnthropic client. Import inside __init__ to avoid import-time dependency.
- `async def complete(...)`: Call `self._client.messages.create()` with system, messages, max_tokens, temperature, model. Map response.usage fields to LLMResponse. Use `getattr(usage, 'cache_creation_input_tokens', 0) or 0` and `getattr(usage, 'cache_read_input_tokens', 0) or 0` for cache fields (safe fallback if SDK version doesn't have them).

**get_provider factory function:**
```python
async def get_provider(session) -> AnthropicProvider:
    """Create an AnthropicProvider using the stored API key."""
    from app.security.vault import FernetVault
    from app.config import get_settings
    vault = FernetVault(get_settings().fernet_key)
    from sqlalchemy import select
    from app.db.models import Secret
    result = await session.execute(select(Secret).where(Secret.name == "anthropic_api_key"))
    row = result.scalar_one_or_none()
    if row is None:
        raise ValueError("Anthropic API key not configured")
    api_key = vault.decrypt(row.ciphertext)
    return AnthropicProvider(api_key=api_key)
```

Use structlog for logging provider creation and call completion (tokens used, cache hit ratio).

Create `app/tailoring/budget.py` with a `BudgetGuard` class:

**Class-level constants:**
- WARN_THRESHOLD = 0.80
- PRICING dict mapping model names to per-MTok rates:
  ```python
  PRICING = {
      "claude-sonnet-4-5": {"input": 3.0, "output": 15.0, "cache_read": 0.30, "cache_write": 3.75},
  }
  ```

**Static methods:**

`estimate_cost(input_tokens, output_tokens, cache_read_tokens=0, cache_write_tokens=0, model="claude-sonnet-4-5") -> float`:
- Calculate cost from token counts using PRICING dict. Return rounded to 6 decimal places.
- Fall back to "claude-sonnet-4-5" pricing if model not in dict.

**Instance methods (takes session):**

`__init__(self)`: Initialize an asyncio.Lock for serialized budget access.

`async def check_budget(self, session) -> tuple[bool, float, float, bool]`:
- Returns (can_proceed, spent, cap, is_warning).
- Load Settings row via get_settings_row.
- Check if budget_month != current month (UTC). If so, reset budget_spent_dollars to 0.0 and update budget_month. Commit.
- If cap <= 0 (no cap set): return (True, spent, 0.0, False).
- is_warning = spent >= cap * WARN_THRESHOLD
- can_proceed = spent < cap
- Return tuple.

`async def debit(self, session, cost_dollars: float, record_id: int, call_type: str, model: str, input_tokens: int, output_tokens: int, cache_read_tokens: int, cache_write_tokens: int)`:
- Acquire the asyncio.Lock.
- Update Settings.budget_spent_dollars += cost_dollars.
- Create a CostLedger row with all the token/cost details.
- Commit.

Use structlog for logging budget checks (spent, cap, can_proceed) and debits.

Per user decision: 80% soft limit (warning banner), 100% hard halt. This method returns the data; the pipeline decides what to do.
