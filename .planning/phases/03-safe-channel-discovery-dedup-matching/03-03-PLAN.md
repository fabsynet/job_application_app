<task type="auto">
  <name>Task 1: Create sources router with CRUD + validation</name>
  <files>app/web/routers/sources.py</files>
  <action>
Create `app/web/routers/sources.py` with APIRouter(prefix="/settings/sources").

Routes:

1. **GET /settings/sources** - Render the sources section partial
   - Load all sources via `get_all_sources(session)`
   - Render `partials/settings_sources.html.j2` with sources list
   - Follow the `_render_section` pattern from settings.py (request, template, context dict)

2. **POST /settings/sources** - Add a new source
   - Accept form field `slug_or_url` (text input)
   - Call `detect_source(slug_or_url)` to parse ATS type and slug
   - Call `validate_source(slug, source_type)` -- hits the real API
   - If "unknown" source_type: probe all three, use first success. If all fail, return inline error
   - If validation fails: return the sources partial with inline error message (flash pattern). Do NOT save
   - If validation succeeds: create Source row via `create_source(session, slug, source_type, display_name)`
   - display_name = slug (can be refined later)
   - Return updated sources list partial (hx-target="#sources-list")
   - Per user decision: "Validate on add -- hit the API immediately. Show inline error if it doesn't resolve. Don't save invalid entries"

3. **POST /settings/sources/{source_id}/toggle** - Toggle enable/disable
   - Read `enabled` from form (checkbox state)
   - Call `toggle_source(session, source_id, enabled)`
   - Return `hx-swap="none"` (HTMX switch toggle, no DOM update needed)

4. **DELETE /settings/sources/{source_id}** - Remove a source
   - Call `delete_source(session, source_id)`
   - Return empty response (hx-swap="outerHTML" removes the row)

Register this router in `app/main.py` via `app.include_router(sources_router.router)`.
